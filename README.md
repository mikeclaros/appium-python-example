# calc-app-appium-example
- Python appium client examples with google calculator

## Requirements
- Android Studio 
- Android device or emulator
- Appium server
- .env file with the following attributes:  
PLATFORM_NAME,  
PLATFORM_VERSION,  
DEVICE_NAME  

## Steps to run
- run appium server
- run calc_app_test.py
